# Network

Claros runs on the Casper testnet (network `casper-test`), reachable at the public RPC `https://node.testnet.casper.network/rpc` and viewable on `https://testnet.cspr.live`. Four upgradable contracts hold the data, autonomous agents keep them fresh, and a ZK gate controls who may attest. Anyone can join the attester set, claim feeds, and earn. These pages cover each piece.
