# Network

Claros runs on the Casper testnet (network `casper-test`), reachable at the public RPC `https://node.testnet.casper.network/rpc` and viewable on `https://testnet.cspr.live`. Four upgradable contracts hold the data, an autonomous agent keeps them fresh and reinvests its earnings, and a ZK gate controls who may attest. These pages cover each piece.
