# Live feeds

Claros publishes 36 live feeds on Casper testnet. Each feed has a stable `feed_id`, and on-chain values are integers: the human value is `amount / 10^decimals` (Pyth-style). Every feed maps to one EIA APIv2 dataset (a route, a frequency, a data column, and a set of facet filters). See [the data catalog](/docs/catalog) for all 232 datasets you can request.

<Callout type="info">
Values are read by `feed_id`. Use [the SDK](/docs/reading/sdk), [the REST API](/docs/reading/rest), or a [cross-contract call](/docs/reading/on-chain) to read them.
</Callout>
